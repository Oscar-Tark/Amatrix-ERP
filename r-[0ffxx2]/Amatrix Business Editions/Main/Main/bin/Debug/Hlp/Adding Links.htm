Adding a Link

 


Amatrix Allows You to Add Links to Your Programs or Files.

 

 

 

Click the My Links Button in the Main Interface or in Amatrix’s Menus, If You Opened the Menu Click on Add a Link…

 

 

                                  

                                                                                                                                                             

The Below Screen will Appear, Click on Next

 

 

 

 

In the Below Shown Screen You may Click on Add and Select any File on Your Computer or Write the Path to a File Manually as an Email address or a Website in the Link Address Text Box

 

 

 

 

Now You May Add a name To Your Link by Writing a Name for Your link in the Link Name Text Box or You may Choose to Use the Original File name of the File you have Chosen by Checking the Use File Name Box

 

 

 

 

Now Click on Finish. You May Now View Your Link by Clicking on the My Links Button’s Arrow  To Open the Links Menu and You Will See Your New Link in the Custom Section
