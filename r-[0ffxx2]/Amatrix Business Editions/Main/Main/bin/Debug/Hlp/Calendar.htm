Using the Utilities : Calendar

 


The Amatrix Calendar Helps You Keep Track of Dates and Times, And Allows You to Set Alarms

 

 

To Start Of Click on the Calendar Button in the Utilities Menu of Amatrix

 

 

 

Now, The Calendar Interface will open As Shown Above

 

 

 

 

You May Change Your Time and Date Settings Through-out Amatrix From Here as to How You Would Like to See Your Time or, Which is the First day of The Week

 

 

 

 

You May Set Alarms Within The Calendar Application, to Set One go To Set Alarm Within the Calendar Interface

 

 

 

 

To Set an Alarm, Set the Time and Date to the Ones You and, Specify a Reason after Which Click on Set. Remember that Alarms only Run when Amatrix is Running.

You May View Previously Set Alarms in the Previous Menu as Shown Below. Amatrix only Has One Alarm
