Using the Amatrix Accounting Studio

 


The Amatrix Accounting Studio Allows You To Manage Accounting And Finances Within Your Business

 

 

·         Accounting Journal

1.        Custom Books

·         Accounting Ledger

·         Invoices

 

Remember, Before Navigating or Searching Save Your Data Unless You Selected Yes in the Use Dynamic Summary/Saving in the Database Menu and

Remember, To Repartition Your Books, You May Find the Repartitioner in the Database Studio or in the Database Menu

 

 

Topic 1 [Accounting Journal]

 

 

To Start Of, You Must Create New Data Entry Element by Entering a Value in an Empty Cell on an Empty Row within any Book, Every Row Has a Serial Number, and You may View 100 Rows at a Time (This Feature was Integrated to Help Users with Older Machines) and Click on Next and Previous to View the Next or Previous 100 Rows. You may also Use the Go to Position before a Command button  to Go to the 100 Rows You Where Viewing before Searching or Running a Data Command

 

 

                                  

                                                                                      

 

You May View Specified Records Which Satisfy Extreme Specifications by Going to the Data Commands Menu and then to the Show Where Menu after Which Choose a Column Name Coincidental to the Main Data View and Select an Operation as Equal To Enter The Data You Want To See and Click on Ok

 

 

 

 

                Sub-Topic [Custom Books]

 

 

In Addition to General Journals, Amatrix Journals Includes a Custom Book Function For Books that may Require More Tailoring by its User. Go to The Custom Books Menu and Click on Open to get Started

 

 

 

 

In the Window Below Click on New Book To Create a New Book (Circled in Red)

 

 

 

 

As The Wizard Shows up, Click on the Next Button

 

 

 

 

Now You May Add a Database that is Located on Your Computer, Or You May Write an SQL Connection String For a Database on a Network an Example is Shown Below

 

"Data Source=Your_Server_Name;Initial Catalog= Your_Database_Name;UserId=Your_Username;Password=Your_Password;" or "Server=Your_Server_Name;Database=Your_Database_Name;UserID=Your_Username;Password=Your_Password;Trusted_Connection=False"

 

If Yours is a Trusted Connection Then :

 

"Data Source=Your_Server_Name;Initial Catalog=Your_Database_Name;Integrated Security=SSPI;" or "Server=Your_Server_Name;Database=Your_Database_Name;Trusted_Connection=True;"

 

Else You May Drop Down the Drop Menu and Choose Internal Data-base to Create Your Book on the Local Database that Amatrix is Using



 

 

You May Also Pick a Name For Your Database so that While Logging in You Wont Get Confused, Do this by Entering a name in the New Name Text Area and Unchecking Do Not Change As Shown Above

 

 

 

 

Now Enter a Password, if Your Database Has One or, If You Dont Have a Password Click on Next and Leave the CheckBox No Password Checked, You May Decide to (only) Add the Database to the Books System by Clicking on the Add Only Data-Base Button as Shown Above

 

 

 

 

The Database Will be Tested to see if it is Compatible with Amatrix, if it is Not You are Free to Leave the Wizard by Clicking on Cancel or Go back and Add a Different Database by Clicking on the Choose a DataBase Tab, if not Click on Next to Continue

 

 

 

 

In the Screen Above, Enter Your Book Name in the Field Shown Below

 

 

 

Now You May Edit the Data Columns That Will Appear In Your Book. Credit and Debit will Be Calculated Automatically

 

 

 

 

Enter a Name of a Column in the Column Name Text Box and, Select a Type of Column in the Type of Column Menu Box for the Type of Data You Would like to Insert into Row Cells on That Specified Column : Numbers, Dates then Click on Add Column. To Remove a Row, Click on The Row as shown Below and Click on Remove

 

 

 

 

After Editing Your Rows You are Ready to Create Your Book, Do So by Clicking on the Below shown Button

 

 

 

 

Now You May Connect To Your Book, Choose a Database and A Book to Connect to and click on Connect
