Using the Utilities : WorkBook

 


The Amatrix WorkBook Is a Simple Text Editor With a Few Useful Utilities for HTML Developers

 

 

To Start Of Click on the WorkBook Button in the Utilities Menu of Amatrix

 

 

 

Now, The WorkBook Interface will open As Shown Below

 

 

 

 

The Amatrix Interface Works The Same as Any Text Editor Interface Except for Its Suggestion system, When it appears (While Typing) Press Page Up or Page Down (PgUp, PgDn) and Use the Arrow Keys to Navigate it
